# Galaxy DeX Toolkit 3.0

Versão focada em investigação e diagnóstico do DeX oculto no Galaxy A56.

## Novidades
- DeX Scanner automático
- Relatório de componentes encontrados/protegidos
- Deteção de monitor externo, resolução e taxa de atualização
- Diagnóstico de rato, teclado e touchpad físico
- Estado da depuração sem fios
- Partilha de relatório técnico
- Correção dos estilos duplicados da v2.4

Abra a pasta no Android Studio e execute **Build > Clean Project** e **Build > Build APK(s)**.
