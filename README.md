# Galaxy DeX Toolkit

![Version](https://img.shields.io/badge/version-v3.0-blue)
![Android](https://img.shields.io/badge/Android-16-green)
![OneUI](https://img.shields.io/badge/One_UI-8.5-blue)

Galaxy DeX Toolkit is an open-source research and diagnostic application created to investigate hidden Samsung DeX components present on Galaxy devices.

## Features

- Hidden Desktop Mode
- Samsung DeX investigation
- Native Touchpad diagnostics
- Input device detection
- Hidden Activities Explorer
- Wireless ADB helper
- Desktop wallpaper tools
- Samsung SystemUI research
- DeX Scanner

## Tested on

- Galaxy A56 (SM-A566B)
- Android 16
- One UI 8.5

## Disclaimer

This application is intended for research and educational purposes only.
It does not modify Samsung firmware or unlock unsupported features.
